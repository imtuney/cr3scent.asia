Selamat Datang di Cr3scent!

Tutorial Bergabung
1
Buka Minecraft Java Edition
Pastikan versi 1.21.1 ke atas. Klik Multiplayer → Add Server.
2
Masukkan Server Address
Paste IP di kolom Server Address:
mc.cr3scent.asia
3
Klik Done & Join
Klik Done, pilih server Cr3scent, lalu klik Join Server. Selesai!
Bedrock wajib pakai port. Kalau ga diisi, tidak akan bisa connect!
1
Buka Minecraft (PE / Console / Win10)
Klik Play → tab Servers → scroll bawah → Add Server.
2
Isi Server Address + Port
Isi dua kolom ini dengan benar:
mc.cr3scent.asia
Port: 25655
3
Save & Connect
Klik Save, pilih server Cr3scent dari list, lalu klik Connect.
4
Register Akun (pertama kali)
Ketik /register [password] [password] di chat untuk mendaftar.
